1. Repetir Stickers (Montaje Pro)

¿Qué hace?

Automatiza la creación de pliegos de impresión para stickers. Es capaz de mezclar diferentes diseños en la misma hoja y calcular cuántos caben de cada uno.

Soporta Máscaras Centrado Auto Mantiene Capas Optimiza Rotación Balance de Cantidades

Pro Tip: No necesitas agrupar las capas de barniz y corte. El script detecta que pertenecen al mismo sticker si están solapadas.

Mini Tutorial

1

Selecciona todos los elementos de tus stickers (incluyendo líneas de corte y barniz). Asegúrate de que cada diseño esté separado del otro.

2

Ejecuta el script. Verás una lista con los diseños detectados y sus medidas reales (ignorando arte oculto por máscaras).

3

Elige "Equilibrar Cantidades" si quieres que el script intente meter la misma cantidad de cada sticker en el pliego.

4

Haz clic en "Calcular" para ver totales y luego en "Generar Montaje" para finalizar.

2. Numeración Secuencial Automática

¿Qué hace?

Transforma cualquier texto seleccionado en un contador inteligente. Ideal para folios, entradas o etiquetas seriadas que requieren un orden específico.

Prefijos/Sufijos Incremento Variable Mantiene Formato Procesamiento Masivo

Consejo: El script mantiene la fuente, el tamaño y el color del texto original. Solo reemplaza el contenido numérico.

Instrucciones

1

Crea un objeto de texto con el formato deseado (ej: "000") y selecciónalo en todos los stickers del montaje.

2

Ingresa el Número Inicial y el Incremento en el panel del script.

3

Pulsa "Numerar". El script detectará la posición de cada texto para asignar los números en orden de lectura.

3. Marcas de Corte Inteligentes

¿Qué hace?

Coloca las marcas de corte tradicionales para guillotina. Su inteligencia evita que las marcas se pisen entre sí cuando los stickers están muy juntos.

Limpieza Automática Modo Sangrado Color Registro Ultra Preciso

Seguridad: Si tus stickers están pegados uno a otro, el script solo pondrá las marcas en los bordes exteriores, dejando el centro limpio.

Mini Tutorial

1

Selecciona el montaje completo de stickers que acabas de generar.

2

Configura el Excedente (Bleed). Si tu sticker ya tiene el fondo más grande, elige la opción "Ya tiene excedente".

3

Mantén activada la "Limpieza inteligente". El script creará la capa "MARCAS" al nivel superior.

4. Registro Graphtec (Suite Pro v9.1)

¿Qué hace?

Genera las marcas de registro exactas para el plotter Graphtec. Incluye herramientas avanzadas de preparación de archivos que no tiene el plugin oficial.

Paso Fijo 300mm Auto-ajuste de Mesa Borde de Fondo 1pt Soporte XY (Tipo 2)

Ahorra tiempo: Pulsa ENTER nada más abrir el script para generar las marcas con la configuración por defecto. ¡Es instantáneo!

Nuevas Funciones

1

Auto-ajustar mesa: El script redimensiona automáticamente el área de trabajo para que el plotter detecte los márgenes de seguridad correctos.

2

Dibujar borde: Crea un rectángulo de 1pt en la capa más profunda. Ideal para archivos que necesitan un fondo sólido exacto.

3

Capa Superior: Las marcas se generan en una nueva capa "GRAPHTEC" por encima de todo el arte.
